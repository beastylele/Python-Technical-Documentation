Introduction

Python is a high-level programming language known for its simplicity and readability. It was created by Guido van Rossum and first released in 1991.

Python emphasizes code readability with its clear and expressive syntax, making it an excellent choice for beginners and experienced programmers alike.

Why is Python useful?

Python is widely used in various industries and domains due to its simplicity, versatility, and robustness.

Common uses of Python:

  • Web Development
  • Machine Learning
  • Data Analysis
  • Artificial Intelligence
  • Automation
Hello World

In Python, printing "Hello, World!" to the screen is straightforward. Here's the code:

print("Hello World!")

Printing in Python is very easy and we don't have to worry about ending lines with a semicolon.

Data Types

Understanding data types is crucial for manipulating and storing different kinds of data in Python programs.

Some commonly used data types:

  • Integers: 3 or 56
  • Floats: 3.1415
  • Strings: "Hello"
  • Booleans: true or false
  • Lists: [1, 2, 3]
  • Dictionaries: {key: value}
User Input

User input allows programs to interact with users by accepting data entered via the keyboard.

In Python, you can use the input() function to prompt the user for input. Here's an example:

name = input("Enter your name: ")
print("Hello", name)

The input() function returns a string containing the user's input, which you can then process further in your program.

You can convert the user input to other data types (e.g., integers, floats) using type conversion functions like int() and float().

Declaring Variables

Variables are used to store data values in a program. In Python, you can declare variables and assign values to them dynamically.

age = 22

Variable names should be descriptive and follow certain naming conventions, such as using lowercase letters and underscores for readability.

user_name = "user123"

Python is dynamically typed, so you don't need to specify the data type when declaring variables.

If/Else Statements

If/else statements are used for decision-making in Python programs. They allow you to execute different blocks of code based on specified conditions.

Conditions are expressions that evaluate to either True or False. If the condition evaluates to True, the code block associated with the if statement is executed. Otherwise, the code block associated with the else statement (if present) is executed.

You can also use elif (short for else if) statements to check multiple conditions sequentially.

if condition:
     print(1)
elif:
     print(2)
else:
     print(3)
Loops

Loops are used to iterate over a sequence of elements or execute a block of code repeatedly until a specified condition is met.

Python supports two main types of loops: for loops and while loops.

  • For loops: These are used for iterating over a sequence (such as a list, tuple, or string)
    • for char in word:
        print(char)
  • While loops: These are used when you don't know in advance how many times the loop should execute.
    • while True:
        print(1)

You can use break to end any loop.

Functions

Functions are blocks of reusable code that perform a specific task. They help in organizing code, improving readability, and reducing redundancy.

In Python, you can define functions using the def keyword followed by the function name and parameters.

Functions can take input arguments, perform operations, and return results using the return statement.

def add(num_one, num_two):
    print(num_one + num_two)
